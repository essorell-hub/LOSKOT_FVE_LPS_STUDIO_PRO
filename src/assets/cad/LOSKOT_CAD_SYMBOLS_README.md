# LOSKOT CAD Symbols

Tato složka je připravená pro SVG/CAD assety programu LOSKOT FVE & LPS STUDIO PRO.

v46 zatím nepřepisuje UI vzhled. Připravuje bezpečný runtime render packet a SVG markup pro FVE CAD panelový náhled.

Další grafická verze může do této složky doplnit:

- FVE panel symboly,
- stringové značky,
- LPS jímače,
- svody,
- HVI vedení,
- SPD značky,
- DC trasy.
