# LOSKOT V86–V90 CAD / Map Deep Integration

MEGA PACK D připravuje datovou/runtime vrstvu pro hlubší integraci CAD/mapy.

## Obsah

- CAD object registry,
- CAD selection engine,
- CAD data link resolver,
- CAD layer manager,
- CAD QA highlight mapper,
- map bridge view model,
- OK/BAD fixtures,
- smoke test,
- verify script,
- statický HTML preview.

## Bezpečnost

Balík nemění Classic PRO UI. Přidává čistou runtime vrstvu do `src/runtime/v86v90`.

## Budoucí přínos

- kliknutí na CAD objekt ukáže data,
- CAD objekt lze propojit s QA,
- vrstvy mají počty objektů,
- vadné datové vazby blokují export,
- mapa/CAD může mít jednotný bridge.
