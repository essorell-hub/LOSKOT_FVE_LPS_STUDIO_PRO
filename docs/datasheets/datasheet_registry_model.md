# Post-Mega Datasheet Registry Model

Purpose: define a safe registry for datasheets before direct document export integration.

## Device types

- pv_module
- inverter
- optimizer
- dc_spd
- ac_spd
- lps_component
- hvi_component
- metering
- switchgear

## Required fields

- id
- deviceType
- manufacturer
- model
- filePath
- source
- status
