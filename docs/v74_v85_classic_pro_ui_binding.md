# LOSKOT V74–V85 Classic PRO UI Binding

Tento MEGA PACK C připravuje UI binding vrstvu bez přímého zásahu do schváleného Classic PRO vzhledu.

## Co přidává

- dashboard view model,
- QA panel view model,
- Project Inspector view model,
- CAD layer view model,
- Document/Export view model,
- Classic PRO app shell state,
- no white screen guard,
- OK/BLOCKED fixtures,
- smoke test,
- verify script.

## Důležité

Toto není přepis UI. Je to bezpečná mezivrstva mezi runtime logikou a budoucím React/Tauri rozhraním.
