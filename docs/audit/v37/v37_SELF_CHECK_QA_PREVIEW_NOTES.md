# v37 SELF CHECK / QA PREVIEW

## Účel

Verze v37 přidává do HTML preview samostatnou obrazovku pro interní kontrolu stavu vývoje programu.

## Co je doplněno

- tlačítko `v37 Kontrola / QA`
- procentuální stav oblastí programu
- QA upozornění a poznámky
- seznam nejbližších nutných prací
- ověřovací skript `scripts/verify/verify_v37_preview.sh`

## Důležité omezení

Procenta jsou technický odhad podle aktuálního vývojového stavu. Nejde o finální release metriku.

## Bezpečnost

v37 vzniká z funkční v36 a nemění starší preview soubory.
