# LOSKOT FVE & LPS STUDIO PRO - v30 Implementační fronta - Datová vrstva

## Úkol: Příprava implementační fronty pro datovou vrstvu (v30)

Tento prompt připravuje další kroky pro implementaci datové vrstvy. Zatím nespouští žádné skripty ani buildy.

### Budoucí implementační fronta:

1.  **`src/data/projectModel.js`** (nebo `.ts`, dle aktuální struktury): Definice sdíleného datového modelu projektu.
2.  **`src/data/loadSampleProject.js`** (nebo `.ts`): Funkce pro načítání vzorových projektů.
3.  **`src/validation/basicProjectValidation.js`** (nebo `.ts`): Základní validace projektu.
4.  **`src/validation/referenceValidation.js`** (nebo `.ts`): Validace referenčních dat.
5.  **`src/importExport/jsonImportPreview.js`** (nebo `.ts`): Náhled importu JSON projektu.
6.  **`src/importExport/jsonExportProject.js`** (nebo `.ts`): Export projektu do JSON.
7.  **`tests/data/basic_validation_smoke_test.js`** (nebo `.ts`): Smoke test pro základní validaci.
8.  **Dokumentace k použití**: Popis a návod k použití nových datových modulů a validací.

### Přísně zakázané akce v této fázi:

*   Jakákoli změna uživatelského rozhraní (UI).
*   Úpravy HTML preview souborů.
*   Změny CSS stylů.
*   Modifikace funkčnosti přepínání obrazovek.
*   Zásahy do vzhledu "Classic PRO".

---

Tento prompt slouží jako příprava pro další iteraci vývoje, zaměřenou na datovou vrstvu.
