# LOSKOT FVE & LPS STUDIO PRO

Sjednocený vývojový repozitář pro jeden společný program spojující:

- LOSKOT FVE Studio PRO
- DEHN / LPS Studio
- Risk Tool
- SPD / LPZ
- CAD / mapa
- dokumenty
- databáze zařízení
- budoucí Windows aplikace

## Hlavní cíl

Vytvořit jeden modulární program pro projektování, kontrolu a dokumentaci FVE, LPS, SPD, LPZ, CAD náhledů, mapových podkladů, databází zařízení, exportů a technických dokumentů.

- Každá zásadní změna se zapisuje do PROJECT_STATE_UNIFIED.md.
